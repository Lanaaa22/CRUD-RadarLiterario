# Radar Literário

## 🔨 Pré Requisitos

- Instalar o Python: https://www.python.org/downloads/

- Instalar o Streamlit:
 ```bash
pip install streamlit
```

- Instalar o psycopg2:
 ```bash
pip install psycopg2-binary
```

## 👾 Compilação e Execução

 ```bash
streamlit run cadastro.py
```

Se o anterior não funcionar, é provável que o Streamlit não esteja no PATH. Então, execute:
 ```bash
python streamlit run cadastro.py
```
